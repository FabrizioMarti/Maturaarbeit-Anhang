# Maturaarbeit-Anhang
Der Kalman-Filter von Kristian Lauszus wurde leicht abgeändert.
Der Grossteil des Journals beinhaltet die Notizen, welche beim Designen des Gerüsts gemacht worden sind. Das Verhalten der Motoren und die Grundprinzipien des Ballbots sind auch 
aufgeschrieben. Im libraries-Ordner ist die selbst geschriebene Library. Um diese zu benutzen, muss man im Libraries-Ordner des Arduino-Dokumentes ein Ordner erstellen und die Library in diese hineinkopieren. Nun kann man wie gewöhnlich mit der #include-Funktion die Library aufrufen und benutzen.
